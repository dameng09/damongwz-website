# 大梦贸易官网 · 快速部署指南（2026-04-27 更新版）

> **预计完成时间：1~2 小时**  
> **预计效果：网站立即可访问，英文版同步上线，询盘表单可用**

## 第1步：注册 Formspree（30分钟）

> 询盘表单需要这个才能收到客户留言

1. 打开浏览器访问：**https://formspree.io**
2. 点击 **Sign Up**，用邮箱注册（免费版每月50条询盘）
3. 注册完成后，点击 **New Form**
4. 填写你的邮箱地址（用于接收询盘通知）
5. 创建后会得到一个 Form ID，格式如：`xabc1234`
6. **备用方案**：如果 Formspree 打不开，用 **https://web3forms.com/**（同样免费，无需注册）

> 📝 **记录下来：** 你的 Form ID 是：`____________`（稍后要填入代码）

---

## 第2步：上传代码到 GitHub（30分钟）

### 2.1 创建 GitHub 仓库

1. 打开 **https://github.com**，登录（没账号先注册）
2. 点击右上角 **"+"** → **"New repository"**
3. 填写：
   - Repository name：`damongwz-website`
   - 选择 **Public**
   - 点击 **Create repository**

### 2.2 上传网站文件

1. 在新建的仓库页面，点击 **"uploading an existing file"**
2. 将以下 **6 个文件/文件夹**全部拖入上传区域：

```
website/
├── index.html        ← 主站（中文）
├── en/
│   └── index.html    ← 英文站（新创建）
├── sitemap.xml       ← 站点地图（已更新）
├── robots.txt        ← SEO文件（新创建）
├── css/
│   └── style.css
└── js/
    └── main.js
```

3. 点击 **Commit changes**

### 2.3 替换 Form ID（重要！）

上传前，先在本地修改 `index.html` 和 `en/index.html`：

**中文版（index.html 第371行）：**
```
找到：action="https://formspree.io/f/YOUR_FORM_ID"
替换为：action="https://formspree.io/f/你的FormID"
```

**英文版（en/index.html 第369行）：**
```
找到：action="https://formspree.io/f/YOUR_FORM_ID"
替换为：action="https://formspree.io/f/你的FormID"
```

## 第3步：部署到 Vercel（15分钟）

1. 打开 **https://vercel.com**
2. 点击 **Sign Up**，用 GitHub 账号登录
3. 点击 **Add New Project**
4. 点击 **Import Git Repository**，找到 `damongwz-website`
5. Framework Preset 选择 **Other**
6. 点击 **Deploy**，等待约 30 秒
7. 看到 **Congratulations!** 表示成功！

---

## 第4步：绑定域名（30分钟，需要你的域名账号操作）

### 4.1 在 Vercel 添加域名

1. 进入项目 → **Settings** → **Domains**
2. 添加两个域名：
   - `damongwz.com`
   - `www.damongwz.com`
3. Vercel 会显示需要配置的 DNS 记录

### 4.2 在域名商配置 DNS

登录你的域名管理后台（阿里云/腾讯云/GoDaddy等），找到 DNS 设置：

| 类型 | 主机记录 | 记录值 |
|------|---------|--------|
| A | @ | `76.76.21.21` |
| CNAME | www | `cname.vercel-dns.com` |

> ⚠️ 具体记录值以 Vercel 控制台显示的为准

**等待时间：** 5分钟 ~ 24小时（大多数 5分钟内生效）

### 4.3 验证

在 Vercel 域名管理页看到绿色 ✅ → 在浏览器输入 `https://www.damongwz.com` 测试访问。
